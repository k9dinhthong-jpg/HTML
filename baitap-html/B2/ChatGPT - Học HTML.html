ChatGPT đã nói:

Bạn cần bắt sự kiện khi bấm nút Login của form. Vì nút của bạn là type="submit" nên thường ta sẽ bắt sự kiện submit của form.

Ví dụ JS đơn giản cho file login.js:

JavaScript
let form = document.querySelector("#login-form");

form.addEventListener("submit", function (event) {
  event.preventDefault(); // chặn form reload trang

  let username = document.querySelector("#username").value;
  let password = document.querySelector("#password").value;

  if (username === "" || password === "") {
    alert("Vui lòng nhập đầy đủ thông tin!");
    return;
  }

  // ví dụ kiểm tra tài khoản mẫu
  if (username === "admin" && password === "123456") {
    alert("Đăng nhập thành công!");
  } else {
    alert("Sai tài khoản hoặc mật khẩu!");
  }
});

Giải thích

  • addEventListener("submit") → bắt sự kiện khi bấm Login.

  • event.preventDefault() → không cho form reload trang.

  • .value → lấy giá trị người dùng nhập.

Luồng hoạt động

  1. Người dùng nhập Email/Phone và Password

  2. Bấm Login

  3. JS kiểm tra:

    • Nếu trống → báo lỗi

    • Nếu đúng → đăng nhập thành công

    • Nếu sai → báo sai tài khoản

  • Cách chuyển trang sau khi login (window.location)

Ví dụ giống code bạn đang làm:

JavaScript
let ClickDangNhap = document.querySelector("button[type='submit']");

ClickDangNhap.addEventListener("click", function (e) {
  e.preventDefault();

  let username = document.querySelector("#username").value;
  let password = document.querySelector("#password").value;

  if (username === "admin" && password === "123456") {
    alert("Đăng nhập thành công");
  } else {
    alert("Sai tài khoản hoặc mật khẩu");
  }
});

Ở đây:

JavaScript
function (e) { ... }

→ chính là function viết trực tiếp trong addEventListener
→ không cần tách ra thành hàm riêng.

Cách viết hiện đại hơn (Arrow Function)

Bạn cũng có thể viết gọn hơn:

JavaScript
let ClickDangNhap = document.querySelector("button[type='submit']");

ClickDangNhap.addEventListener("click", (e) => {
  e.preventDefault();
  console.log("Đăng nhập");
});

So sánh 2 cách

1️⃣ Viết trực tiếp (thường dùng cho code ngắn)

JavaScript
addEventListener("click", function(){})

2️⃣ Tách hàm ra (dùng khi code dài)

JavaScript
function dangNhap() {
  console.log("Login");
}

button.addEventListener("click", dangNhap);

💡 Với form login nhỏ như bài của bạn, dev thường viết trực tiếp trong sự kiện cho nhanh.
